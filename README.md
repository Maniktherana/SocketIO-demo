## How to install

Clone this repo

```bash
git clone https://github.com/Maniktherana/SocketIO-demo.git
```

## How to run

### Frontend

```Bash
cd client
npm install
npm run dev
```

### Backend

```Bash
cd server
npm install
npm run dev
```
